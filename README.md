# Tablet Bazzar — POS móvil Bazzar

PWA Next.js para vendedores en las 6 tiendas. **Report** monitorea; esta app ejecuta.

## Inicio rápido

```bash
cd tablet-bazzar
npm install
# .env.local con DATABASE_URL
npm run dev   # http://localhost:3002
```
